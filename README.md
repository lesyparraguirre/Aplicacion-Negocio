# Aplicacion-Negocio
Se requiere una aplicación de escritorio destinada a la gestión integral de productos y ventas para un comercio. El sistema deberá ser genérico, de forma que pueda adaptarse a distintos tipos de negocios, y su información podrá ser utilizada luego por otros servicios o herramientas administrativas según sea necesario.

La aplicación permitirá administrar productos y realizar el proceso completo de venta, incluyendo la generación de la boleta final y el cálculo del costo de envío.

Funcionalidades principales

Gestión de productos

Listado de productos.

Búsqueda por diferentes criterios.

Alta de productos.

Modificación de productos existentes.

Eliminación de productos.

Visualización del detalle completo de cada producto.

Carga y modificación de precios, imágenes y descripción.

Proceso de venta y generación de boleta

Selección de productos a vender.

Cálculo automático del total.

Generación de boleta final con detalle de la compra.

Gestión de método de pago, incluyendo:

Pago en efectivo (con cálculo automático del vuelto).

Pago mediante Mercado Pago (con registro/importación del monto exacto).

Cálculo de envío

El sistema debe permitir calcular el costo de envío a partir de los kilómetros de distancia ingresados por el usuario.

La tarifa por kilómetro deberá ser configurable.

El costo final de envío debe sumarse automáticamente a la boleta.

Gestión de usuarios

Usuario Administrador: con permisos completos para gestionar productos, usuarios, valores de envío, precios, etc.

Usuario Normal: acceso limitado únicamente a la carga de ventas y generación de boletas.

Persistencia de datos

Toda la información deberá ser almacenada y gestionada mediante una base de datos, permitiendo mantener registro de productos, ventas, métodos de pago, usuarios y configuración de tarifas.
